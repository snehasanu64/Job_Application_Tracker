# 📋 Job Application Tracker

A simple client-side job application tracker built while actively job hunting.
Track every company you apply to, update the status as things progress, and never lose track of an application again.

## ✨ Features

- Add applications with company name, role, date applied
- Track status — Applied → Interview → Rejected → Offer
- Filter applications by status
- Clean, responsive UI
- No login needed — runs entirely in the browser
